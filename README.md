# 🧮 Python CLI Calculator

A full-featured command-line calculator built with Python — my first Python project!

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📐 Modulus (Remainder)
- 🔢 Power (x ^ y)
- √  Square Root
- |x| Absolute Value
- %  Percentage Calculator
- 📊 Logarithm (base 10)
- 📜 Calculation History
- 🗑️ Clear History

## 🚀 How to Run

Make sure you have Python 3 installed, then run:

```bash
python calculator.py
```
